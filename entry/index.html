WebXR Unity Render Streaming

This page allows you to stream a Unity VR application from the Cloud into your VR headset via WebXR. This WebXR Client (built using A-Frame) integrates with the Open Source VR Streaming Unity SDK. To test this CloudXR streaming solution, follow the README instructions linked on Github.

Please note that this SDK is still experimental and in active development. If you run into any issues, please create a Github Issue or ask on Discord.
